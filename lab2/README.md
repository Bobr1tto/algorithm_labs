# Лабораторная работа №2
## Маршрут автобуса
Реализована система управления маршрутом автобуса

Каждая остановка содержит:
- Название остановки
- Координаты (x, y)
- Время до следующей остановки

Реализовано:
- Добавление остановки (вставка рядом с ближайшей по координатам)
- Расчёт общего времени маршрута
- Определение, где будет автобус через N остановок
- Расчёт времени прибытия
- Построение обратного маршрута
- Сохранение маршрута в Base64-файл и загрузка из него

## Описание функций

### `add_stop(stop)` — добавление остановки
Добавляет новую остановку в маршрут. Если маршрут пуст — ставит первой. Если одна остановка — добавляет в конец. Иначе ищет ближайшую по координатам и вставляет рядом с ней. Время между остановками рассчитывается автоматически через расстояние.
```python
route.add_stop(Stop("Вокзал", Position(0, 0)))
route.add_stop(Stop("Центр", Position(10, 0)))
```

### `get_total_time()` — общее время маршрута
Проходит по всем узлам списка и суммирует время каждого перегона. Возвращает число минут от первой до последней остановки.
```python
print(route.get_total_time())
```

### `find_stop_after_n(from_title, n, depart_time)` — где автобус через N остановок
Находит стартовую остановку по названию, затем делает N шагов вперёд по списку, считая время в пути. Если передать `depart_time`, дополнительно выводит расчётное время прибытия.
```python
from datetime import datetime

route.find_stop_after_n("Вокзал", n=3)

depart = datetime(2025, 1, 1, 8, 0)
route.find_stop_after_n("Вокзал", n=3, depart_time=depart)
```

### `show()` — вывод маршрута
Печатает все остановки по порядку с координатами и временем до следующей. В конце выводит общее время маршрута.
```python
route.show()
```
```
Маршрут:
   1. Вокзал               (    0,     0)  →  [7 мин]
   2. Центр                (   10,     0)  →  [13 мин]
   3. Парк                 (   20,     0)  (конечная)

  Общее время маршрута: 20 мин
```

### `reverse()` — обратный маршрут
Разворачивает весь список: последняя остановка становится первой. У каждого узла меняются местами ссылки `nxt` и `prv`, время перегонов перераспределяется в обратном порядке.
```python
route.reverse()
route.show()
```

### `save_to_file(filename)` — сохранение в файл
Сохраняет маршрут в текстовый файл. Каждая остановка кодируется в Base64. Строка до кодирования имеет вид `Название|x|y|время`, после — набор символов Base64.
```python
route.save_to_file("route.txt")
```
Пример содержимого файла:
```
0JLQvtC60LfQsNC7fDB8MHw3
0KbQtdC90YLRgHwxMHwwfDEz
0J/QsNGA0LuxMjB8MHww
```

### `load_from_file(filename)` — загрузка из файла
Читает файл, декодирует каждую строку из Base64, восстанавливает объекты остановок и собирает список заново. Текущий маршрут перед загрузкой очищается.
```python
route2 = RouteList()
route2.load_from_file("route.txt")
route2.show()
```
