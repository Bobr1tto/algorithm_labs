# Дерево выражений

Строим дерево из постфиксной записи и вычисляем его рекурсивно.

## Запуск

```bash
python main.py
```

## Функции

- `build_expression_tree(tokens)` — строит дерево из токенов постфиксной записи
- `evaluate(root)` — рекурсивно вычисляет выражение
- `constant_folding(root)` — схлопывает узлы с числами в готовый результат

## Пример

```
Постфикс: 2 5 * 3 +

└── +
    ├── *
    │   ├── 2
    │   └── 5
    └── 3

Результат: 13
После folding: 13
```
