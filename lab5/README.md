# Дерево выражений (Обратная польская запись)

Построение дерева выражения из постфиксной записи с рекурсивным вычислением на чистом Python.

---

## Запуск

```bash
python main.py
```

---

## Структура узла

Каждый узел — обычный словарь:

```python
{'value': '+', 'left': node, 'right': node}  # внутренний узел
{'value': 5,   'left': None, 'right': None}  # лист (операнд)
```

---

## Функции

| Функция | Описание |
|---|---|
| `make_node(value, left, right)` | Создаёт узел дерева |
| `is_leaf(node)` | Проверяет, является ли узел листом |
| `build_expression_tree(tokens)` | Строит дерево из списка токенов постфиксной записи |
| `evaluate(node)` | Рекурсивно вычисляет значение дерева |
| `fold_constants(node)` | Сворачивает константные подвыражения (constant folding) |
| `print_tree(node)` | Выводит дерево в терминал |

---

## Алгоритм построения

Используется стек. Токены читаются слева направо:
- **число** → кладётся в стек как лист
- **оператор** → снимаются два верхних узла, создаётся новый узел с ними как детьми, кладётся обратно

В конце в стеке остаётся один элемент — корень дерева.

---
