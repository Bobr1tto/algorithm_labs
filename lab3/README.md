# Система обработки задач (Queue)

Реализация очереди задач на Python. Работает по принципу **FIFO** (First In, First Out) — первая добавленная задача выполняется первой.

## Запуск

```bash
python queue_tasks.py
```

---

## Классы

### `Task`

Представляет одну задачу.

| Атрибут | Тип | Описание |
|---|---|---|
| `id` | `int` | Уникальный идентификатор задачи |
| `description` | `str` | Описание задачи (по умолчанию `""`) |
| `priority` | `int` | Приоритет задачи (по умолчанию `1`) |

**Пример:**
```python
task = Task(1, "Исправить баг", 3)
```

---

### `Queue`

Очередь задач. Хранит список объектов `Task` и управляет ими.

#### Методы

**`add_task(task)`**
Добавляет задачу в конец очереди.
```python
queue.add_task(Task(1, "Обновить зависимости", 2))
```

**`remove_task()`**
Удаляет и возвращает первую задачу из очереди. Если очередь пуста — возвращает `None`.
```python
queue.remove_task()
```

**`peek()`**
Возвращает первую задачу, не удаляя её. Если очередь пуста — возвращает `None`.
```python
first = queue.peek()
print(first.description)
```

**`is_empty()`**
Возвращает `True`, если очередь пуста, иначе `False`.
```python
if queue.is_empty():
    print("Задач нет")
```

**`find_by_id(id)`**
Проверяет, есть ли в очереди задача с указанным `id`. Возвращает `True` или `False`.
```python
queue.find_by_id(2)   # True
queue.find_by_id(99)  # False
```

**`show_tasks()`**
Выводит все задачи в очереди в виде нумерованного списка.
```python
queue.show_tasks()
```

Пример вывода:
```
Задачи:
1. Обновить зависимости проекта (приоритет = 2, id = 1)
2. Написать документацию (приоритет = 1, id = 2)
3. Исправить баг в авторизации (приоритет = 3, id = 3)
```
